# الخطيب للحدادة العامة | Al-Khatib Blacksmithing

خبرة في الحدادة العربية التقليدية والأوروبية الحديثة، الزخارف المعدنية والنقش بالليزر في عمان، الأردن.

## التقنيات المستخدمة

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## كيفية التشغيل محلياً

اتبع الخطوات التالية لتشغيل المشروع على جهازك:

```sh
# 1. تثبيت التبعيات
npm install

# 2. تشغيل خادم التطوير
npm run dev
```

## بناء المشروع للإنتاج

```sh
npm run build
```
